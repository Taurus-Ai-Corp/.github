### Licensing

We use a **Triple-Plus Licensing Model**:

- **MIT** - Core utilities (free, open source)
- **FSL** - Community packages (free, non-commercial)
- **BSL** - Professional features ($99-299/mo)
- **Commercial** - Enterprise deployment ($999-4,999/mo)
- **White-Label** - OEM partnerships ($50K+ setup)
